## 5문답

### 1. JDK, JRE, JVM의 차이

- JVM은 자바 바이트코드를 실행하는 가상 머신이다.
- JRE는 JVM과 자바 실행에 필요한 라이브러리를 포함한 실행 환경이다.
- JDK는 JRE에 더해 `javac` 같은 개발 도구까지 포함한 개발 키트다.

### 2. 클래스와 객체의 차이

- 클래스는 객체를 만들기 위한 설계도다.
- 객체는 클래스를 바탕으로 실제 메모리에 생성된 인스턴스다.
- 예를 들어 `Room`이 클래스라면, `1번 스터디룸`은 객체다.

### 3. 배열과 ArrayList의 차이

- 배열은 길이가 고정되어 있고 같은 타입의 데이터를 저장한다.
- `ArrayList`는 크기를 동적으로 늘리거나 줄일 수 있다.
- 배열은 기본 문법 수준의 자료구조이고, `ArrayList`는 메서드가 풍부해서 실무에서 더 자주 다룬다.

### 4. HTTP 요청-응답 흐름을 5단계로 설명

1. 브라우저가 URL 요청을 보낸다.
2. 서버는 요청을 받고 어떤 자원이 필요한지 해석한다.
3. 애플리케이션이 비즈니스 로직을 수행하고 필요하면 DB를 조회한다.
4. 서버가 결과를 HTTP 응답 형태로 만든다.
5. 브라우저가 응답을 받아 화면에 렌더링하거나 후속 동작을 수행한다.

### 5. 같은 시간대 중복 예약이 왜 발생할 수 있는지

- 두 요청이 거의 동시에 들어오면 둘 다 "아직 예약되지 않았다"라고 판단할 수 있다.
- 검증과 저장이 원자적으로 처리되지 않으면 같은 슬롯이 중복 저장될 수 있다.
- 그래서 이후 단계에서 트랜잭션, 락, 동시성 제어가 중요해진다.
